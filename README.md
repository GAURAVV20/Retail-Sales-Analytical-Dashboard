# Retail Sales Analytics Dashboard (Power BI)

This project is an interactive **Retail Sales Analytics Dashboard** built in Power BI using the Global Superstore dataset.  
It analyzes revenue, profit, and profitability across **regions, product categories, and customer segments** and presents key business insights in a single-page dashboard.
